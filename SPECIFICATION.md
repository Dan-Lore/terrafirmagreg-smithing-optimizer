# Спецификация проекта: «Точка G (минимальные команды)»

## 1. Постановка задачи
Пользователь вводит целое число `G`, где `0 <= G <= 150`. Требуется получить число `G`, начиная с `0`, применяя последовательность команд.

Набор доступных команд (всегда статический):
- `+2`, `+7`, `+13`, `+16`
- `-3`, `-6`, `-9`, `-15`

Количество применённых команд должно быть **минимальным** (оптимизация по длине).

Ограничение по состояниям: на каждом шаге текущее значение должно оставаться в диапазоне `0..150`.

## 2. Результат для пользователя
Для введённого `G` приложение показывает:
- `steps`: минимальное количество шагов (равно длине последовательности)
- `moves`: последовательность команд (дельты), которая начиная с `0` приводит к `G` и при этом ни разу не выходит за пределы `0..150`.

Если путь не существует при заданном ограничении — отображается “недостижимо”.

## 3. Алгоритм предвычисления
Так как:
1) набор команд фиксирован,
2) целевые точки `G` покрывают небольшой диапазон `0..150`,
3) требуется один раз заранее посчитать ответы,

используется предвычисление:

1) строится невзвешенный ориентированный граф состояний `x` на вершинах `0..150`;
2) из состояния `x` есть рёбра в `y = x + delta`, где `delta` — одна из 8 команд, при условии `0 <= y <= 150`;
3) запускается BFS от `0`, чтобы для всех достижимых `G` получить кратчайшие пути по количеству шагов;
4) для восстановления пути сохраняются:
   - `prev[v]`: предыдущее состояние
   - `move[v]`: команда (дельта), которой пришли в `v`

Восстановленный путь для каждого `G` — это массив `moves`, который можно применить по порядку начиная с `0`.

## 4. Формат сгенерированных данных
Сгенерированный файл:
- `src/generated/paths.ts`

Экспорт:
- `PATHS`: массив длины 151 (индексы соответствуют значениям `g` от 0 до 150).

Тип:
- `PATHS[g] = null`, если состояние недостижимо;
- иначе `PATHS[g] = { steps: number; moves: number[] }`.

## 5. Запуск генерации
Скрипт:
- `scripts/precompute-paths.ts`

Команда:
- `npm run precompute`

Этот скрипт один раз проходит по всем состояниям и перезаписывает `src/generated/paths.ts`.
